# TX_simulator
TX simulator to know how often Tsukuba Express run.

Black : Nomal train <br>

Red: Rapid train<br>

Blue : Section fast train<br>




# How to use
Download all files

Install "Processing" here https://processing.org/download/

Open "Processing" and open the program below
